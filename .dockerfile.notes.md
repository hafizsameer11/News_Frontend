# Docker Build Notes

## Environment Variables

Set these environment variables when running the container:

```bash
docker run -p 3000:3000 \
  -e NEXT_PUBLIC_API_URL=https://your-backend-api.com \
  -e NEXT_PUBLIC_FRONTEND_URL=https://your-frontend-domain.com \
  your-image-name
```

Or use a `.env` file:
```bash
docker run -p 3000:3000 --env-file .env your-image-name
```

## Required Environment Variables

- `NEXT_PUBLIC_API_URL` - Your backend API URL (e.g., `https://api.example.com`)
- `NEXT_PUBLIC_FRONTEND_URL` - Your frontend URL (e.g., `https://www.example.com`)

## Optional Environment Variables

- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` - Stripe publishable key
- `NEXT_PUBLIC_TINYMCE_API_KEY` - TinyMCE API key
- `NEXT_PUBLIC_CDN_URL` - CDN URL for images
- `NEXT_PUBLIC_BYPASS_PAYMENT` - Payment bypass flag
- Social media URLs (Facebook, Twitter, Instagram, etc.)
- Contact information (email, phone)

## Building the Image

```bash
docker build -t news-next-frontend .
```

## Running the Container

```bash
docker run -p 3000:3000 \
  -e NEXT_PUBLIC_API_URL=http://localhost:3001 \
  -e NEXT_PUBLIC_FRONTEND_URL=http://localhost:3000 \
  news-next-frontend
```
